# SkillPilot (Python-only)

MVP карьерного помощника для хакатона. Всё на Python.
Запуск:
1) python -m venv .venv && (Linux/Mac) source .venv/bin/activate  |  (Win) .venv\Scripts\activate
2) pip install -r requirements.txt
3) cp .env.example .env  (по желанию добавьте OPENAI_API_KEY)
4) python run.py
